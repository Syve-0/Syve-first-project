# 剧本杀店铺管理系统

## 项目介绍

本项目是一个基于 Java Web 技术栈开发的剧本杀店铺管理系统，采用 Maven 构建，使用 MySQL 数据库存储数据。系统支持三种角色：管理员（ADMIN）、主持人/DM（DM）和玩家（PLAYER），提供剧本管理、场次调度、用户管理、财务统计、组局大厅、DM工作台等核心功能。前端采用现代化的 UI 设计，包含登录注册、动态菜单、数据可视化等交互功能。

## 使用方法

1. 配置数据库连接信息，确保 MySQL 服务已启动，并导入项目所需的数据库表结构
2. 使用 Maven 构建项目（`mvn clean package`），然后将生成的 WAR 文件部署到 Tomcat 服务器，或直接通过 IDE 运行项目，访问 `http://localhost:8080/Login.jsp` 进行登录
